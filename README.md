# 图片字幕生成器

一个基于Web的图片字幕生成工具，可以为图片添加多行字幕，每行字幕具有独立的背景块，呈现切割感效果。

## 功能特性

- ✅ 图片上传（支持JPG、PNG、GIF等常见格式）
- ✅ 字幕设置（高度、字体大小、颜色、轮廓、字体样式、粗细）
- ✅ 多行字幕输入
- ✅ 每行字幕独立背景块（切割感效果）
- ✅ 实时预览
- ✅ 图片下载保存

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"选择图片"按钮上传图片
3. 在"字幕设置"中调整字幕样式参数
4. 在"字幕内容"文本框中输入字幕（每行对应图片上的一行字幕）
5. 点击"生成字幕图片"按钮生成带字幕的图片
6. 在右侧预览区域查看效果
7. 点击"保存图片"按钮下载生成的图片

## 技术实现

- **HTML5 Canvas API**: 用于图片处理和字幕渲染
- **纯JavaScript**: 无框架依赖，可直接运行
- **响应式设计**: 支持不同屏幕尺寸

## 核心特性说明

### 每行字幕独立背景块

这是本工具的核心特性：
- 每行字幕文字都有独立的背景矩形块
- 背景块之间有明显间距，形成切割感
- 背景块宽度根据文字内容自动调整
- 背景块颜色为半透明黑色（rgba(0, 0, 0, 0.6)）
